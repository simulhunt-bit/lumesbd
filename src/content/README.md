# Content Editing Guide

Update brand text in `src/content/brand.ts`.

Update homepage event sections separately in:
- `src/content/events/event-one.ts`
- `src/content/events/event-two.ts`

Update categories, subcategories, and products in `src/content/catalog.ts`.

Each product includes:
- price
- stock
- sizes
- colors
- images
- story slides

Replace placeholder product images in `public/assets/products/` with real photos later.

Site-level brand assets:
- Favicon is loaded from `src/app/icon.png`
- Main shared logo image for metadata/social previews is `public/lumes-logo.png`
